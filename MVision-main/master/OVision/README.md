# OVision

This project was generated with [Angular CLI](https://github.com/angular/angular-cli) version 16.2.6.

## Environmental Settings
OVision relies on communicating with the VLink server, this address is defined in /src/shared/environments/environments.ts
If attempting to access OVision outside of the master server, this address must be set to master_server_ip:3000 rather than localhost
